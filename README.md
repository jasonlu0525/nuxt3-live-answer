# Day 24 - 實作預約房型功能 解答

## 安裝

以下將會引導你如何安裝此專案到你的電腦上。

Node.js 版本建議為：`18.0.0` 以上

### 取得專案

````bash
git clone -b day24-booking  https://github.com/jasonlu0525/nuxt3-live-answer.git day24-booking-answer

### 移動到專案內

```bash
cd day24-booking-answer
````

### 安裝套件

```bash
npm install
```

### 運行專案

```bash
npm run dev
```

### 開啟專案

在瀏覽器網址列輸入以下即可看到畫面

```bash
http://localhost:3000/
```

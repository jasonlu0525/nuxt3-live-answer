# Day 7 - 嵌套式路由、useRouter & useRoute 解答

## 安裝

以下將會引導你如何安裝此專案到你的電腦上。

Node.js 版本建議為：`18.0.0` 以上

### 取得專案

```bash
git clone -b [branch]  https://github.com/jasonlu0525/nuxt3-live-answer.git [fileName]
```

### 移動到專案內

```bash
cd [fileName]
```

### 安裝套件

```bash
npm install
```

### 運行專案

```bash
npm run dev
```

### 開啟專案

在瀏覽器網址列輸入以下即可看到畫面

```bash
http://localhost:3000/
```
